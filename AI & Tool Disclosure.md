### **AI & Tool Disclosure**

In accordance with the 3MTT NextGen Knowledge Showcase guidelines, I am disclosing the tools and Artificial Intelligence models utilized in the development of the AL-IKHTIRA platform:

1. **Google Gemini 3.0 Flash API:** Used as the core cognitive engine for the application. It powers the dynamic extraction of concepts, generation of educational content, and mathematical evaluation within the LangGraph agents.  
2. **LangGraph & LangChain:** Used to structure the multi-agent architecture and manage conversational memory states.  
3. **Streamlit:** Used as the Python framework to rapidly develop and deploy the frontend user interface.  
4. **AI Coding Assistants:** Large Language Models (Gemini) were utilized during the development phase as a pair-programming assistant to help architect the LangGraph state machine, debug routing loops, and format the Python code efficiently. All pedagogical logic and system architecture designs were directed by me.
